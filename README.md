# worksession_skill

A powerful Amazon Alexa skill that allows you to focus while working/studying.

### Simply say:
- "Alexa, start work sesh", and a 50 minute focused work session will begin.

ENABLE IT HERE: [amazon.com](https://cutt.ly/kfawRin)
